RTOS / Real-time Control Integration
Turning AI inference results into reliable real-time actions (alerts, IO control) requires state management and debouncing.
把 AI 結果變成可靠的即時控制：狀態管理、去抖動、失效保護。
Control Loop (Concept)
  • Input: AI events (e.g., detections / confidence / states)
  • Decision: state machine (normal → warning → alarm)
  • Output: MCU IO control (buzzer / LED) + OSD status
Reliability Patterns
  • Debounce / hysteresis: avoid flickering alerts when confidence oscillates.
  • Rate limiting: cap alert frequency to prevent user fatigue.
  • Fail-safe: define behavior when inference is invalid / no signal / model failure.
  • Logging: minimal event log for field debugging.
可靠性：去抖動/遲滯、頻率限制、失效保護、最小化 log 方便現場 debug。
